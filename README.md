# 🎓 Radar-Acad – Sistema de Gestão Acadêmica

Este projeto web simula um sistema completo de gerenciamento escolar e acadêmico, permitindo o controle absoluto sobre a estrutura de ensino. É uma excelente aplicação para consolidar conhecimentos em desenvolvimento web full-stack, modelagem de banco de dados relacional e design de interfaces modernas (UI/UX).

## 🎯 Funcionalidades
- **Cadastros Base:** Gerenciamento de Áreas do Saber, Ocupações, Turnos e Cidades.
- **Estrutura Acadêmica:** Cadastro e vinculação de Pessoas (Alunos/Professores), Instituições, Cursos, Turmas e Disciplinas.
- **Controle Diário:** Registro de frequências e ocorrências disciplinares.
- **Interface Moderna:** Design responsivo com tema escuro (Dark Mode) e efeitos de "Glassmorphism".
- **Painel Administrativo:** Área restrita para gestão rápida de todos os dados do sistema.

## 🛠️ Tecnologias
- Python (Django Framework)
- HTML5 e CSS3 customizado
- Bootstrap 5
- SQLite (Banco de Dados)
